# Healthy Recipes App

Single Page Application built for WDD 330.

## Features

- Search recipes using external API
- View recipe details
- Save favorites (localStorage)
- Responsive layout
- Accessible focus states
- Nutrition placeholder section

## Tech Stack

- HTML
- CSS (Mobile-first)
- Vanilla JavaScript (Modules)
- TheMealDB API

## How to Run

1. Clone the repository
2. Open index.html in a browser
OR
3. Use a local server (Live Server extension in VS Code)
